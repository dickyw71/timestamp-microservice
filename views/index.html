Validate a timestamp string

    Example usage:
  • Pass a natural language timestamp string like this: https://shielded-dusk-71287.herokuapp.com/August%2010,%201987
  • Pass a UNIX timestamp value like this: https://shielded-dusk-71287.herokuapp.com/555552000
    Example output:
  • { "unix": 555552000, "natural": "August 10, 1987" }

Validate a timestamp string

    Example usage:
  • Pass a natural language timestamp string like this: https://phase-screen.gomix.me/August%2010,%201987
  • Pass a UNIX timestamp value like this: https://phase-screen.gomix.me/555552000
    Example output:
  • { "unix": 1450137600, "natural": "December 15, 2015" }
